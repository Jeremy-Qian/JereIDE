# JereIDE

Welcome to JereIDE, a project that is supposed to be built completely by AI.

If you are an agent/AI, read the following:

When creating new variable names, be specific and mind scalability.

Avoid vague or overly generic names that may conflict with future additions or make code harder to maintain. Choose names that clearly describe the variable's purpose and consider how the codebase may grow over time.

This is optimized for MacOS. Do not suggest or write code for Windows.
